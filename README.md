# 2dsA-app-mar-Bank
O MaréBank é uma aplicação de banco digital desenvolvida para incentivar a educação financeira entre adolescentes e jovens. Seu objetivo é ajudar os usuários a controlar seus gastos, criar hábitos de economia e aprender sobre planejamento financeiro de forma simples, prática e interativa.

Funcionalidades
- Categorização automática de gastos.
- Cofres para criação de metas financeiras.
- Alertas de limite de gastos.
- Cartão mesada com controle pelos responsáveis.
- MaréBank Academy com conteúdos educativos, desafios e jogos sobre educação financeira.
  
Tecnologias Utilizadas

Frontend: HTML e CSS
Backend: Django
Banco de Dados: PostgreSQL
Objetivo

Promover o controle financeiro entre os jovens, reduzindo gastos impulsivos e incentivando o planejamento financeiro por meio da tecnologia e da educação.

Público-Alvo
Adolescentes e jovens.
Pais e responsáveis.
Escolas interessadas em educação financeira.
Resultados Esperados

Com o MaréBank, espera-se que os usuários desenvolvam hábitos financeiros mais saudáveis, aprendam a economizar, acompanhem seus gastos e tomem decisões mais conscientes sobre o uso do dinheiro.
